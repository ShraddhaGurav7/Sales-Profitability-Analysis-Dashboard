# Sales-Profitability-Analysis-Dashboard
## Project Overview
This project analyzes sales data to evaluate business performance, profitability, and the impact of discount strategies using MySQL and Power BI.

---

## Tools & Technologies
- MySQL (Data Analysis)
- Power BI (Visualization)
- Excel (Data Source)

---

## Key Features
- KPI tracking (Sales, Profit, Profit Margin, Quantity)
- Sales vs Profit comparison across categories
- Discount impact analysis using scatter plots
- Region and segment-based performance analysis
- Identification of loss-making sub-categories

---

## Key Insights
- Higher discounts negatively impact profitability  
- Some sub-categories generate losses despite high sales  
- Sales volume does not always translate into profit  
- West region performs best in terms of sales  

---

## Business Impact
This dashboard helps stakeholders:
- Identify loss-making products  
- Optimize pricing strategies  
- Improve decision-making through data insights
